Введение в Infor CRM Back Office Extension

Infor CRM Back Office Extension (ICBOE) добавляет следующие сущности в мобильный клиент:
  • Счета
  • Предложения
  • Заказы на продажу
  • Доставки
  • Дебиторская задолженность
Для отображения этих сущностей необходимо активировать следующие представления:
  • Выберите глобальное меню > Другое > Настройка меню, нажмите Счета, Дебиторская задолженность, и/или Доставки. В верхней панели меню нажмите значок с флажком, чтобы применить сделанный выбор, и выйдите из настройки меню.
  • При необходимости задайте группу по умолчанию для сущности: в списке сущностей выберите На задана группа по умолчанию. Нажмите здесь, чтобы настроить группы и укажите, требуется ли отображать все доступные записи для данной сущности или только ваши записи для этой сущности.

  • Вкладки "Связанные элементы" подробного представления сущности предоставляет доступ к дополнительной информации. Откройте подробное представление для следующих типов сущностей, чтобы получить доступ к элементам на вкладке "Связанные элементы":

    • Счета: "Элементы счета" и "Элементы дебиторской задолженности".
    • Предложения: "Строки предложения", "Вложения", "Продавцы" и "История синхронизации".
    • Дебиторская задолженность: "Элементы дебиторской задолженности"
    • Заказы на продажу: "Строки заказа", "Строи счета", "Строки доставки", "Вложения" и "История синхронизации"
    • Доставки: "Строки доставки"

Быстрые подсказки

  • Группы доступны для предложений и заказов на продажу.
  • Кнопка "Удалить" доступна в элементах строки "Предложение" и "Заказ". Нажмите ее, чтобы удалить текущую запись из CRM. Эта функция доступна только в подробном виде элемента строки.
  • При работе в автономном режиме обе функции Недавно просмотренные и Мой показать доступны для записей ICBOE.

Информационные панели

Информационные панели предназначены для отображения настраиваемой, предварительно заданной аналитики по определенному клиенту в виде виджетов со списком.

Выберите в подробностях клиента вкладку Информационные панели.

  • Финансовый снимок: отображает информацию о настраиваемом периоде в 30, 60, 90, 180 и 365 дней.
    • Доход по выставленным счетам: общая сумма для счетов, относящихся к клиенту и созданных в течение последних x дней.
    • Прибыль по выставленным счетам: (общая сумма) за вычетом (общей стоимости) для счетов, относящихся к клиенту и созданных в течение последних x дней.
    • Маржа по выставленным счетам: (сумма прибыли по выставленным счетам) деленная на (общая сумма) для счетов, относящихся к клиенту и созданных в течение последних x дней.
    • Доход в годовом исчислении: (доход за прошлый год за вычетом дохода в период 2 лет и 1 предшествующего года) деленная на (доход в период 2 лет и 1 предшествующего года).
    • Прибыль в годовом исчислении: (общая сумма) за вычетом (общей стоимости) для счетов, относящихся к клиенту и созданных в течение последних x дней.
    • Маржа в годовом исчислении: (маржа за прошлый год за вычетом маржи в период 2 лет и 1 предшествующего года) деленная на (маржу в период 2 лет и 1 предшествующего года) для счетов, относящихся к клиенту и созданных в течение последних x дней.
  • Новые транзакции: отображает данные за настроенный период в 7, 14, 30 и 90 дней, который включает текущую дату. Отображаются как количество счетов, так и их общая сумма.
    • Новые предложения: количество предложений, относящихся к клиенту и созданных в течение последних x дней.
    • Новые заказы: количество заказов, относящихся к клиенту и созданных в течение последних x дней.
    • Новые доставки: количество доставок, относящихся к этому клиенту и созданных в течение последних x дней.
    • Новые счета: количество счетов, относящихся к клиенту и созданных в течение последних x дней.
  • Открытые транзакции: отображает открытые устаревшие предложения, устаревшие счета и устаревшие заказы на продажу за настраиваемый период в 0+, 30+, 60+, 90+ и 180+ дней до текущей даты. Отображаются как количество записей транзакций, так и их общая сумма.
    • Устаревшие предложения: общее количество открытых предложений, относящихся к клиенту и созданных в течение последних x дней.
    • Устаревшие счета: общее количество открытых счетов, относящихся к клиенту и созданных в течение последних x дней.
    • Устаревшие заказы на продаж: общее количество открытых заказов на продажу, относящихся к этому клиенту и созданных в течение последних x дней.

Поля поиска ICBOE

Следующие поля поиска доступны, если в установленной системе Infor CRM имеется интеграция Infor CRM Back Office Extension. Перейдите на вкладку справки Infor CRM, чтобы просмотреть список дополнительных полей поиска.

Для каждой сущности результаты поиска можно отфильтровать по перечисленным ниже параметрам.

  • Клиенты

    Поля поиска

    • ID заказчика в ERP: начинается с

  • Клиенты цели счета

    Поля поиска

    • Имя цели счета: начинается с

  • Контакт

    Поля поиска

    • Имя контакта: начинается с

  • Клиент контакта

    Поля поиска

    • Имя контакта: начинается с

  • Счета

    Поля поиска

    • Имя клиента: начинается с
    • ID счета: начинается с

  • Элементы счета

    Поля поиска

    • Имя продукта: начинается с
    • ID счета: начинается с
    • Описание продукта: начинается с
    • Номер строки: начинается с

  • Предложения

    Поля поиска

    • Номер предложения: начинается с
    • Имя клиента: начинается с
    • ID заказа в ERP: начинается с

  • Строки предложения

    Поля поиска

    • Номер строки: начинается с
    • Имя продукта: начинается с
    • Описание продукта: начинается с
    • Номер предложения: начинается с

  • Ответственные за предложения лица

    Поля поиска

    • Имя: начинается с
    • ID предложения: начинается с

  • Дебиторская задолженность

    Поля поиска

    • ID дебиторской задолженности: содержит
    • Имя клиента: содержит
    • ID счета: содержит

  • Элементы дебиторской задолженности

    Поля поиска

    • ID счета: содержит
    • ID дебиторской задолженности: содержит
    • Имя клиента: содержит

  • Заказы на продажу

    Поля поиска

    • Номер заказа на продажу: начинается с
    • Имя клиента: начинается с
    • ID заказа в ERP: начинается с
    • Заказ на покупку заказчика: начинается с

  • Строки заказа на продажу

    Поля поиска

    • Номер строки: начинается с
    • Имя продукта: начинается с
    • Описание продукта: начинается с
    • Номер заказа на продажу: начинается с

  • Доставки
