快速提示

  • 点击此图标会转至“全局”菜单，从而可执行以下操作：
    • 使用 SpeedSearch 执行搜索（适用于 Saleslogix 8.0 和更高版本）。
    • 使用快速操作可快速添加客户/联系人。
    • 使用转至可访问移动设备客户端中的所有记录类型。
    • 使用配置菜单可更改“转至”菜单中的项，并指定这些项的显示顺序。“配置菜单”屏幕上列表中的第一项将成为默认视图，会在您登录后首先加载。
    • 使用设置可清除保存的凭据、清空存储以及访问错误日志。
    • “设置”中的“清空存储”选项是严格用于解决问题的一种措施，因为此操作会清空本地存储的配置。
    • 查找帮助。
    • 注销。
    • 连接显示您的移动设备客户端是处于联机状态（记录中的标题栏将显示为蓝色）还是处于脱机状态（标题栏将显示为灰色）。
  • 点击此图标会转至上下文（右侧）菜单。可用选项因记录类型而异。
    • 可用的井号标签
    • 关键绩效指标(KPI)
    • 展开组 -> 配置以选择要在列表视图上显示的组。这适用于“客户”、“联系人”、“销售线索”、“销售机会”和“工单”列表视图。
    • 选择一个布局。这适用于“客户”、“联系人”、“销售线索”、“销售机会”和“工单”列表视图。以后，当您再次返回到此记录类型时，会默认采用所选的布局。
      • 摘要布局在 1 列中显示 4 个字段。这是默认设置。
      • 详细信息布局根据设备的屏幕宽度在 1 到 3 列中显示字段。
        注意：如果您具有 CRM Web 或 Windows 客户端的访问权，那么可以修改某些字段在这些“移动摘要”和“详细信息”布局中的显示方式。
        • 从 Web 客户端的“客户”、“联系人”、“销售线索”、“销售机会”或“工单”列表视图中，右键单击组选项卡，然后单击编辑，接着单击布局选项卡：
        • 拖放各列，调整它们的顺序。
        • 以绿色文本显示货币字段：选择潜力列，单击编辑，展开格式类型字段下拉列表，然后选择货币，单击确定。
        • 将电话号码显示为可点击的链接：选择电话号码，单击编辑，展开格式类型字段下拉列表，然后选择电话号码，最后单击确定。
        查看 Web 或 Windows 客户端帮助，了解有关编辑组的更多信息。
  • 此图标表示可展开和折叠面板，以显示/隐藏额外的详细信息。
  • 此图标表示可以向 CRM 添加新记录。这适用于所有列表视图。
  • 如果可以使用“刷新”按钮，那么点击该按钮会立即刷新信息。选择一条记录，以使刷新操作限于此记录。如果未选择任何记录，那么会刷新整个组的数据。
  • 详细信息视图中的蓝色文本表示链接操作（例如，浏览至 URL 或跳至某个客户）。
  • 注意：使用 iPhone 或 iPad 时，可从主屏幕上的一个快捷方式访问 Infor CRM 移动应用，“映射”链接将在新视图中打开，而 Web 链接将被禁用。

快速操作

  • 在列表视图中，点击记录图标虚拟形象可查看和访问常用功能的快捷方式。可用选项将因记录类型而异。（部分快速操作图标可能不可用。例如，在“我的活动”中，如果已接受会议，那么“拒绝”操作将不可用。）“我的活动”和“附件”在 Saleslogix 8.0 和更高版本中提供。
  • 使用配置以更改“快速操作”窗格中项目的可见性和顺序。

正在脱机工作

    当您的移动设备客户端处于脱机状态或失去连接时，可通过三个功能进行只读访问，以选择记录和实体。
  • 最近查看显示访问过的任何详细视图，因此它们在失去连接时可用。“最近查看”在全局菜单上提供。
  • 通过我的公文包可以选择在脱机时可用的记录。要将项目添加到“我的公文包”，请在详细视图中打开一条记录，然后单击标题栏中的公文包图标。该记录的所有相关数据也会移至公文包中。从“我的公文包”视图打开一条记录后，标题栏将显示为灰色，因为公文包始终处于脱机模式。
  • 我的日程安排显示当天的活动。“我的日程安排”在全局菜单上提供。您可以将“我的日程安排”列表视图中的所有活动都移入“我的公文包”中。要执行此操作，请打开我的日程安排列表视图，然后单击标题栏中的公文包图标。
    • 注意：“脱机工作”在 Windows 电话 8 上不受支持。

搜索

  • 可通过多种方式执行搜索：

    • 从全局菜单，查找所有相关的客户、联系人、销售线索、销售机会、活动、备注或历史以及工单。输入更多文本以提高搜索的准确性。
    • 从上下文菜单，使用预定义的井号标签关键字过滤列表视图结果。
    • 对于每个实体，可通过查找栏使用下面指定的字段过滤列表视图结果。

  • 客户

    搜索字段

    • 客户名称：开头为

  • 联系人

    搜索字段

    • 名：开头为
    • 姓：开头为
    • NameLF：包含

  • 销售线索

    搜索字段

    • 公司名称：开头为
    • 名：开头为
    • 姓：开头为
    • NameLF：包含

  • 工单

    搜索字段

    • 编号：开头为

    • 客户名称：开头为
