Présentation de Infor CRM Back Office Extension

Infor CRM Back Office Extension (ICBOE) ajoute les entités suivantes au client mobile :
  • Factures
  • Devis
  • Bons de commande
  • Expéditions
  • Créances
Pour voir ces entités, vous devez activer les vues :
  • Cliquez sur menu Global > Autre > menu Configurer, cliquez sur Factures, Créances et/ou Expéditions. Dans la barre de menus supérieure, cliquez sur la coche pour accepter les sélections et quitter le menu Configurer.
  • Si nécessaire, définissez un groupe par défaut pour une entité : Dans la vue Liste des entités, cliquez sur Aucun groupe par défaut défini. Cliquez ici pour configurer des groupes et sélectionnez l'option appropriée pour voir les enregistrements disponibles pour l'entité ou uniquement vos propres enregistrements pour cette même entité.

  • Les onglets Eléments associés des vues de détails des entités donnent accès à des informations supplémentaires. Ouvrez une vue de détails pour les types d'entité suivants afin d'accéder aux éléments de l'onglet Eléments associés :

    • Facturation : éléments de facture et de créance.
    • Devis : lignes de devis, pièces jointes, représentants commerciaux et historique de la synchronisation.
    • Créances : éléments de créance
    • Bons de commande : lignes de commande, lignes de facture, lignes d'expédition, pièces jointes et historique de la synchronisation
    • Expéditions : lignes d'expédition

Conseils rapides

  • Les groupes sont disponibles pour les devis et les bons de commande.
  • Le bouton Effacer est disponible dans les éléments de ligne Devis et Commande. Appuyez dessus pour supprimer l'enregistrement actuel de CRM. Uniquement disponible dans la vue Détails de la ligne.
  • Les fonctions Affiché récemment et Mon porte-documents sont disponibles en travail hors ligne pour les enregistrements ICBOE.

Tableaux de bord

Utilisez les tableaux de bord pour voir les analyses prédéfinies et personnalisables relatives aux comptes sous la forme de widgets de liste.

Sur la présentation Détails du compte, cliquez sur l'onglet Tableaux de bord.

  • Résumé financier : affiche les informations sur une période configurable de 30, 60, 90, 180 et 365 jours.
    • Chiffre d'affaires annuel facturé : montant total étendu des factures associées au compte, créées dans les x derniers jours.
    • Profit facturé : (montant total étendu) moins le (coût total) pour les factures associées au compte, créées dans les x derniers jours.
    • Marge facturée : (montant du profit facturé) divisé par le (montant étendu) pour les factures associées au compte, créées dans les x derniers jours.
    • Taux de variation chiffre d'affaires (annuel) : (chiffre d'affaires de l'année passée moins le chiffre d'affaires précédent entre deux ans et un an) divisé par le (chiffre d'affaires précédent entre deux ans et un an).
    • Profit sur un an : (montant total étendu) moins le (coût total) des factures associées au compte, créées dans les x derniers jours.
    • Taux de variation marge (annuel) : (marge de l'année passée moins la marge précédente entre deux ans et un an) divisée par la (marge précédente entre deux ans et un an) pour les factures associées au compte, créées dans les x derniers jours.
  • Nouvelles transactions : affiche les transactions sur une période configurable de 7, 14, 30 et 90 jours comprenant la date actuelle. Affiche également le numéro et la valeur totale des factures.
    • Nouveaux devis : nombre de devis associés au compte, créés dans les x derniers jours.
    • Nouveaux bons de commande : nombre de bons de commande associés au compte, créés dans les x derniers jours.
    • Nouvelles expéditions : nombre d'expéditions associées au compte, créées dans les x derniers jours.
    • Nouvelles factures : nombre de factures associées au compte, créées dans les x derniers jours.
  • Transactions en cours : affiche les devis, les factures et les bons de commandes en cours existants sur une période configurable de 0+, 30+, 60+, 90+ et 180+ jours précédents la date actuelle. Affiche également le nombre des enregistrements de transaction et leur valeur totale.
    • Devis existants : nombre total de devis en cours associés au compte, créés dans les x derniers jours.
    • Factures existantes : nombre total de factures en cours associées au compte, créées dans les x derniers jours.
    • Bons de commande existants : nombre total de bons de commande en cours associés au compte, créés dans les x derniers jours.
