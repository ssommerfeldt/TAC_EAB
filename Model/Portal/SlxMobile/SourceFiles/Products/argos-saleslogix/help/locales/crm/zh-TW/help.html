  • 點選此圖示移至「全域」功能表，以進行下列操作：

    • 使用移至以存取 Mobile Client 中的所有記錄類型。
    • 使用設定功能表變更「移至」功能表中的項目，並指定項目顯示的順序。「設定功能表」畫面的第一個清單項目將成為預設檢視，並在登入後首先載入。
    • 使用設定可清除儲存的認證、清除儲存空間及存取錯誤日誌。
    • 「設定」下方的「清除儲存空間」選項只是一個疑難排解裝置，用於清除儲存在本機的設定。
    • 尋找說明。
    • 登出。
    • 連線顯示您的 Mobile Client 為線上 (記錄中的標頭列將會顯示為藍色) 或離線 (標頭列將會顯示為灰色) 狀態。
  • 點選此圖示移至「操作」(右邊) 功能表。可用選項會因記錄類型而所有不同。
    • 可用的 # 標籤
    • 關鍵效能指標 (KPI)
    • 展開群組、組態，以選取要在清單檢視上顯示的群組。此功能可在帳戶、聯絡人、銷售線索、商機和記錄單清單檢視上使用。

      • 摘要配置會在 1 欄顯示 4 個欄位。這是預設值。
      • 詳細資料配置會根據裝置的寬度，顯示 1 至 3 欄的欄位。
        附註：如果您有 CRM 網頁或 Windows 用戶端的存取權，可以修改這些行動摘要和詳細資料配置中一些欄位的顯示方式。
        • 在 Web Client 客戶、聯絡人、銷售線索、商機或記錄單清單檢視的群組標籤上按一下滑鼠右鍵，按一下編輯，再按一下配置標籤：
        • 拖放欄以變更其順序。
        • 以綠色文字顯示貨幣欄位：選取潛力欄，按一下編輯，展開格式類型欄位下拉式清單，然後選取貨幣，按一下確定。
        • 將電話號碼顯示為可按一下的連結：選取電話號碼，按一下編輯，展開格式類型欄位下拉式清單，然後選取電話號碼，按一下確定。
        查看網頁或 Windows 用戶端說明，以取得編輯群組的詳細資訊。

工作離線

    有三個功能可讓您在 Mobile Client 離線或是遺失連線時唯讀存取以選取記錄和項目。
  • 最近檢視顯示造訪過的任何詳細資料檢視，即使連線遺失仍然可以使用這些檢視。在全域功能表中可以使用最近檢視。
  • 我的公事包可讓您選取能夠在離線時可以使用的記錄。若要將項目新增至「我的公事包」，請在詳細資料檢視中開啟一個記錄，然後按一下標題列中的公事包圖示。該記錄的所有相關資料也會被移至公事包。當您從「我的公事包」檢視中開啟記錄時，由於「公事包」始終處於離線模式，因此標題列會是灰色的。
  • 標頭我的排程顯示今日的活動。我的排程可以透過全域功能表取得。您可以將我的排程清單檢視中的所有活動移入我的公事包中。若要執行此動作，請開啟我的排程清單檢視然後按一下標頭列中的公事包圖示。
    • 備註: Windows Phone 8 中不支援工作離線。

搜尋

  • 搜尋的方法有很多種:

    • 在「全域」功能表中找尋所有相關客戶、聯絡人、銷售線索、商機、活動、備註/歷程記錄和記錄單。輸入較多文字以提高搜尋的準確性。
    • 在「操作」功能表，使用預先定義的 # 標籤關鍵字，篩選清單檢視的結果。
    • 針對每個實體，在查閱列使用下方指定的欄位篩選清單檢視結果。
