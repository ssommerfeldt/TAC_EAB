Infor CRM Back Office Extension 簡介

Infor CRM Back Office Extension (ICBOE) 將下列實體新增至 Mobile Client:
  • 發票
  • 報價
  • 銷售訂單
  • 貨運
  • 應收帳款
若要查看這些實體，您必須啟用檢視:
  • 依序按一下全域功能表 > 其他 > 設定功能表，點選發票、應收帳款和/或貨運。在頂端功能表列中，按一下勾選記號圖示以接受選擇並且退出設定功能表。
  • 如果需要，設定實體的預設群組: 在實體清單檢視中，按一下未設定預設群組。按一下此處以設定群組並且選擇查看該實體的所有可用記錄或是只查看該實體中您的記錄。

  • 實體詳細資料檢視相關項目標籤提供存取其他資料。開啟下列實體類型的詳細資料檢視以存取相關項目標籤中的項目:

    • 發票: 發票項目與應收帳款項目。
    • 報價: 報價行、附件、銷售人員與同步歷程記錄。
    • 應收帳款: 應收帳款項目
    • 銷售訂單: 訂單行、發票行、貨運行、附件和同步歷程記錄
    • 貨運: 貨運行

快速提示

  • 可用於報價和銷售訂單的群組。
  • 「報價」與「訂單」行項目中可以使用刪除按鈕，點選此按鈕可刪除 CRM 中的目前記錄。此按鈕只能在行項目的詳細資料檢視中使用。
  • 工作離線最近檢視和我的公事包功能可以在 ICBOE 記錄中使用。

儀表板

使用儀表板，以清單小組件形式查看客戶相關的可自訂、預定義分析。

在客戶詳細資料配置中，按一下儀表板標籤。

  • 財務快照: 以可設定的 30、60、90、180 與 365 天的期間顯示資訊。
    • 發票收益: 過去 x 天中與建立的帳戶相關發票的總計擴充金額。
    • 發票盈利: 過去 x 天中與建立的帳戶相關發票的 (總計擴充金額) 減去 (總成本)。
    • 發票利潤：過去 x 天中與建立的帳戶相關發票的 (發票盈利金額) 除以 (擴充金額)。
    • YoY 收益: (過去一年的收益減去 2 年前和 1 年前之間的收益) 除以 ( 2 年前和 1 年前之間的收益)。
    • YoY 盈利: 過去 x 天中與建立的帳戶相關發票的 (總計擴充金額) 減去 (總成本)。
    • YoY 利潤: 過去 x 天中與建立的帳戶相關發票的 (過去一年的利潤減去 2 年前和 1 年前之間的利潤) 除以 (2 年前和 1 年前之間的利潤)。
  • 新增交易: 顯示可設定的 7、14、30 以及 90 天期間 (包括目前日期)。會顯示發票數目及其總值。
    • 新增報價: 過去 x 天中與建立的帳戶相關的報價數目。
    • 新增訂單: 過去 x 天中與建立的帳戶相關的訂單數目。
    • 新增貨運: 過去 x 天中與建立的帳戶相關的貨運總目。
    • 新增發票：過去 x 天中與建立的帳戶相關的發票數目。
  • 未結交易: 顯示目前日期之前可設定的 0+、30+、60+、90+ 以及 180+ 天期間的未結時效報價、時效發票以及時效銷售訂單。將會顯示交易記錄數目及其總值。
    • 時效報價: 過去 x 天中與建立的帳戶相關的未結報價總數。
    • 時效發票: 過去 x 天中與建立的帳戶相關的未結發票總數。
    • 時效銷售訂單: 過去 x 天中與建立的帳戶相關的未結銷售訂單總數。

ICBOE 搜尋欄位

    搜尋欄位
