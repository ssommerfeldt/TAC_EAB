Conseils rapides

  • Activez cette icône pour accéder au menu Global pour :
    • Effectuer une recherche avec SpeedSearch (Saleslogix 8.0 et versions ultérieures).
    • Ajouter rapidement un compte ou un contact grâce aux actions rapides.
    • Accédez à tous les types d'enregistrement du client mobile grâce à la fonction Accéder.
    • Sélectionnez Configurer le menu pour modifier les éléments du menu Accéder et définir leur ordre d'affichage. Le premier élément de la liste de l'écran Configurer le menu devient votre vue par défaut et est chargé en premier une fois que vous êtes connecté.
    • Accédez aux paramètres pour effacer les identifiants enregistrés ou le stockage, ainsi que pour accéder aux journaux d'erreurs.
    • L'option « Effacer le stockage » de la section Paramètres est un simple outil de dépannage permettant d'effacer la configuration stockée en local.
    • Accédez à l'aide.
    • Déconnectez-vous.
    • Connexion indique si votre client mobile est en ligne (la barre d'en-tête des enregistrements s'affiche en bleu) ou hors ligne (la barre d'en-tête s'affiche en gris).
  • Appuyez sur cette icône pour ouvrir un menu contextuel (à droite). Les options disponibles varient selon le type d'enregistrement.
    • Mots clés disponibles
    • Indicateurs de performance clé (KPI)
    • Développez les commandes Groupes, Configurer pour sélectionner les groupes à afficher dans vos vues Liste. Ces commandes sont disponibles dans les vues Liste suivantes : Compte, Contact, Prospect, Opportunité et Ticket.
    • Sélectionnez une présentation. Ces commandes sont disponibles dans les vues Liste suivantes : Compte, Contact, Prospect, Opportunité et Ticket. La présentation sélectionnée devient la présentation par défaut lorsque vous réaffichez ce type de fiche.
      • La présentation Résumé se caractérise par quatre champs dans une colonne. Il s'agit du paramètre par défaut.
      • La présentation Détails contient 1 à 3 colonnes selon la largeur de votre périphérique.
        Remarque : si vous avez accès au client CRM Web ou Windows, vous pouvez modifier l'affichage de certains champs dans les présentations Résumé et Détails pour mobiles.
        • Dans la vue Liste Compte , Contact, Prospect, Opportunité ou Ticket du client Web, cliquez avec le bouton droit de la souris sur un onglet de groupe, cliquez sur Modifier, puis sur l'onglet Présentation :
        • Changez l'ordre des colonnes en glissant à l'emplacement souhaité.
        • Affichage en vert du texte des champs de devise : sélectionnez la colonne voulue, cliquez sur Modifier, développez la liste déroulante associée au champ Type de format et sélectionnez Devise, puis cliquez sur OK.
        • Affichage des numéros de téléphone sous la forme de liens actifs : sélectionnez le numéro de téléphone, cliquez sur Modifier, développez la liste déroulante associée au champ Type de format et sélectionnez Numéro de téléphone, puis cliquez sur OK.
        Pour de plus amples informations sur la modification des groupes, reportez-vous à l'aide des clients Web et Windows.
  • Cette icône indique que vous pouvez développer et réduire les volets afin d'afficher ou de masquer les détails.
  • Cette icône indique que vous pouvez ajouter un nouvel enregistrement dans CRM. Elle est disponible dans toutes les vues Liste.
  • Lorsque le bouton Actualiser est disponible, appuyez dessus pour actualiser immédiatement les informations. Si vous souhaitez que l'actualisation ne s'applique qu'à un seul enregistrement, sélectionnez ce dernier. Lorsqu'aucun enregistrement n'est sélectionné, ce sont les données de l'ensemble du groupe qui sont actualisées.
  • Le texte en bleu dans les vues Détails indique une action hypertexte (par exemple, l'accès à une URL ou à un compte).
  • Note : Quand vous utilisez un iPhone ou un iPad et que vous lancez le client mobile Infor CRM à partir d'un racourci de l'écran d'accueil, le lien Plan s'ouvre sur une nouvelle vue et le lien Web est désactivé.

Actions rapides

  • Dans les vues Liste, cliquez sur l'icône de la fiche pour accéder aux raccourcis des fonctions fréquentes. Les options disponibles varient selon le type d'enregistrement (certaines icônes d'actions rapides peuvent ne pas être disponibles. Par exemple, dans Mes activités, l'action Refuser ne sera pas disponible si vous avez déjà accepté la réunion). Les sections Mes activités et Pièces jointes sont disponibles dans Saleslogix8.0 et versions ultérieures.
  • Sélectionnez Configurer pour modifier la visibilité et l'ordre des éléments du volet Action rapide.

Travail hors ligne

    Trois fonctions pemettent l'accès en lecture seule à certains enregistrements quand votre client mobile est hors ligne ou perd la connectivité.
  • Affiché récemment affiche toutes les vues de détails visitées, et disponibles en cas de perte de connectivité. Affiché Récemment est accessible à partir du menu Global.
  • Mon porte-documents vous permet de sélectionner les enregistrements qui seront disponibles hors ligne. Pour ajouter des éléments à Mon porte-documents, sélectionnez la vue Détails, ouvrez un enregistrement, puis cliquez sur l'icône de porte-documents située sur la barre d'en-tête. Toutes les données associées à cet enregistrement sont ensuite déplacées dans le Porte-documents. Quand un enregistrement est ouvert dans la vue Mon porte-documents, la barre d'en-tête est grisée car le Porte-documents est toujours en mode hors ligne.
  • Ma planification affiche les activités d'aujourd'hui. Ma planification est accessible depuis le menu Global. Vous pouvez déplacer toutes les activités de la vue Liste Ma planification dans Mon porte-documents. Pour ce faire, ouvrez la vue Liste Ma planification et cliquez sur l'icône Porte-documents située sur la barre d'en-tête.
    • Note: Le travail hors ligne n'est pas pris en charge sous Windows Phone 8.

Rechercher

  • La recherche peut s'effectuer de différentes manières :

    • Dans le menu Global, vous pouvez effectuer une recherche sur tous les comptes associés, contacts, leads, opportunités, activités, notes/historique et tickets. Saisissez davantage d'informations pour affiner votre recherche.
    • Dans le menu contextuel, vous pouvez filtrer les résultats de la vue Liste avec des mots clés prédéfinis
    • Pour chaque entité, filtrez les résultats de la vue Liste depuis la barre de recherche à l'aide des champs spécifiés ci-dessous.
