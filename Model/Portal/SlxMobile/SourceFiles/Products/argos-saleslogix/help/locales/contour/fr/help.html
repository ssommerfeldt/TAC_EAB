Utiliser Proximité

La fonction de recherche de proximité est utilisée pour rechercher des comptes situés dans un rayon de 800 Km autour d'un emplacement.
  • Rechercher des comptes autour d'un point de départ spécifique.
  • Rechercher des comptes à proximité en utilisant un compte comme point de départ.

Trouver les fonctions de recherche de proximité

  • A partir du menu Global :
    • Activez les Comptes à proximité pour identifier des comptes situés à proximité de vos coordonnées GPS. Utiliser la liste déroulante Type de compte pour filtrer les résultats par suspect, lead, prospect, client, partenaire, vendeur, prescripteur, concurrent ou autre. Les résultats de la recherche s'affichent dans la vue Details du compte.
    • Activez la Recherche de proximité pour identifier des comptes situés à proximité d'emplacement spécifiques (entreprises, repères, etc.). Ces emplacements sont paramétrables dans le menu Emplacements de Infor CRM et peuvent inclure les emplacements partagés par d'autre utilisateurs. Si vous avez renseigné les champs Mon bureau et Mon domicile dans votre profil utilisateur Infor CRM, ils apparaîtront dans les options possibles. Utiliser la fonction Recherche pour saisir une adresse ou la sélectionner dans la liste des emplacements disponibles.
  • A partir de la vue Détail du compte :
    • Dans la barre d'en-tête de la vue Détail du compte, activez Comptes à proximité pour lancer une recherche à partir du compte sélectionné.
