介绍 Infor CRM Back Office Extension

Infor CRM Back Office Extension (ICBOE) 添加下列实体到移动设备客户端：
  • 发票
  • 报价单
  • 销售订单
  • 发货
  • 应收帐款
若要查看这些实体，您必须启用下列视图：
  • 单击全局菜单> 其他 > 配置菜单，点击 发票，应收帐款，和/或 发货。在顶部的菜单栏中单击勾选标记图标，接受选择，退出“配置菜单”。
  • 若有必要，请为实体设置默认组：在实体列表视图中，单击无默认组设置。 单击此处来配置组，选择查看该实体所有可用的记录或只查看该实体中您的记录。

  • 通过实体详细视图“相关项”选项卡，可访问其他信息。打开详细视图，使下列实体类型可访问“相关项”选项卡中的项目：

    • 发票：发票项目和应收帐款项目。
    • 报价单：报价单行、附件、销售人员和同步历史。
    • 应收帐款：应收帐款项
    • 销售订单：订单行、发票行、发货行、附件和同步历史
    • 发货：发货行

快速提示

  • 组可用于报价单和销售订单。
  • “删除”按钮在“报价单”和“订单”行项目上提供，点击该按钮会将当前记录从 CRM 中删除。这仅适用于行项目的详细视图。
  • 脱机工作下最近查看和我的公文包功能对 ICBOE记录均可用。

仪表板

使用“仪表板”，在列表小组件表单中查看与客户相关的可自定义的预定义分析。

在“客户”详细信息布局中单击仪表板选项卡。

  • 财务快照：显示可配置的 30、60、90、180 和 365 天期间的信息。
    • 已开发票收入： 与过去 x 天内创建的客户相关的发票的“总金额”。
    • 已开发票利润： 与过去 x 天内创建的客户相关的发票的（总金额）减去（总成本）。
    • 已开发票利润率： 与过去 x 天内创建的客户相关的发票的（已开发票利润额）除以（总金额）。
    • 收入逐年增长率： 与过去 x 天内创建的客户相关的发票的(去年与今年的收入差减去两年前和一年前的收入差)除以（两年前和一年前的收入差）。
    • 逐年利润增长： 与过去 x 天内创建的客户相关的发票的（总金额）减去（总成本）。
    • 逐年利润率增长率： 与过去 x 天内创建的客户相关的发票的（去年与今年的利润差减去两年前和一年前的利润差）除以（两年前和一年前的利润差）。
  • 新事务： 显示包括当前日期在内的可配置的 7、14、30 和 90 天的期间。将显示发票数量及其总值的数量。
    • 新报价单： 与过去 x 天内创建的客户相关的报价单的数量。
    • 新订单： 与过去 x 天内创建的客户相关的订单的数量。
    • 新发货： 与过去 x 天内创建的客户相关的发货的数量。
    • 新发票： 与过去 x 天内创建的客户相关的发票的数量。
  • 未结事务： 显示在包括当前日期在内的可配置的 7、14、30 和 90 天的期间内的未结“报价单数量”、“发票数量”和“销售订单数量”。将显示事务记录及其总值的数量。
    • 报价单总数： 于过去 x 天内创建的客户相关的未结报价单合计的数量。
    • 发票总数： 于过去 x 天内创建的客户相关的未结发票合计的数量。
    • 销售订单总数： 于过去 x 天内创建的客户相关的未结销售订单合计的数量。
